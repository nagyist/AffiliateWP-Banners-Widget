== Description ==

With this plugin you'll be able to easily display nice [AffiliateWP](http://affiliatewp.com/?ref=35) banners from the affiliate system via a Widget.

== Installation ==

1. Upload the entire 'affiliatewp-banners-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Appearance" then, "Widgets"
4. Configure the built-in Widget
